## 📁 三、文件命名规范

### 命名规则
- 使用小写字母
- 单词之间用连字符 `-` 分隔
- 避免使用空格、中文或特殊字符
- 保持简洁且具有描述性

### 标准命名示例
```
✅ 正确示例：
- profile.jpg
- soul-1.jpg
- soul-2.jpg
- soul-3.jpg

❌ 错误示例：
- 个人照片.jpg
- Soul 1.jpg
- SOUL_2.JPG
- my photo (1).png
```

## 💡 六、额外建议

### 图片质量建议
1. **分辨率**：使用推荐尺寸的2倍（如 1280x1600）以支持高清屏幕
2. **文件大小**：单张图片控制在 500KB 以内，保证加载速度
3. **色彩**：照片色彩饱和度适中，避免过度曝光或过暗
4. **构图**：生活照片建议有一定故事性或艺术感

### 文字内容建议
1. **长度**：每段正文建议 50-150 字，保持简洁
2. **语言**：可以中英文混合，但保持风格统一
3. **关键词**：在自我介绍中突出你的专业领域和特色
4. **语气**：保持真诚、自然的个人风格

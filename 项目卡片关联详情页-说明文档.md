# 如何关联项目卡片和详情页

## 📋 完整流程说明

### 步骤1：创建项目详情页文件

为每个项目创建一个详情页文件。文件路径决定了访问URL。

**示例：**
```
文件路径：app/projects/ai-trading/page.tsx
访问URL：http://localhost:3001/projects/ai-trading

文件路径：app/projects/data-viz/page.tsx
访问URL：http://localhost:3001/projects/data-viz
```

**操作步骤：**
1. 在 `app/projects/` 目录下创建新文件夹（用英文命名，如 `ai-trading`）
2. 在新文件夹中创建 `page.tsx` 文件
3. 复制 `app/projects/example/page.tsx` 的内容到新文件
4. 修改内容（见下方）

---

### 步骤2：编辑项目详情页内容

打开你创建的 `page.tsx` 文件，修改以下内容：

```typescript
import { ProjectDetailLayout, ProjectSection, ProjectImage } from "@/components/project-detail-layout"

export default function AITradingPage() {  // 修改函数名
  // 1. 定义章节列表
  const sections = [
    { id: "overview", title: "项目概述" },
    { id: "background", title: "项目背景" },
    { id: "features", title: "核心功能" },
    { id: "results", title: "项目成果" },
  ]

  return (
    <ProjectDetailLayout
      title="AI量化交易系统"              // 修改项目标题
      date="2024.12.15"                   // 修改日期
      tags={["Python", "AI", "Trading"]}  // 修改标签
      githubUrl="https://github.com/..."  // 修改GitHub链接（可选）
      sections={sections}
    >
      {/* 2. 编写章节内容 */}
      <ProjectSection id="overview" title="项目概述">
        <p>
          这里写你的项目介绍...
        </p>
      </ProjectSection>

      <ProjectSection id="background" title="项目背景">
        <p>
          项目背景描述...
        </p>
        <ProjectImage
          src="/images/projects/ai-trading-1.jpg"
          alt="项目截图"
          caption="图1：系统界面"
        />
      </ProjectSection>

      {/* 继续添加其他章节... */}
    </ProjectDetailLayout>
  )
}
```

### 步骤3：在项目列表中添加链接

打开 `components/projects-section.tsx`，找到 `projects` 数组，修改或添加项目：

```typescript
const projects = [
  {
    title: "AI量化交易系统",                    // 项目标题
    description: "基于机器学习的智能交易系统...", // 项目描述
    tags: ["Python", "AI", "Trading"],          // 技术标签
    year: "2024",                               // 年份
    featured: true,                             // 是否精选（大卡片）
    image: "/images/projects/ai-trading.jpg",   // 封面图片
    link: "/projects/ai-trading",               // ⭐ 关键：详情页链接
  },
  {
    title: "数据可视化平台",
    description: "企业级数据可视化解决方案...",
    tags: ["React", "D3.js", "TypeScript"],
    year: "2024",
    featured: true,
    image: "/images/projects/data-viz.jpg",
    link: "/projects/data-viz",                 // ⭐ 对应详情页路径
  },
  // ... 更多项目
]
```

**重要：** `link` 字段的值必须与详情页的文件路径对应：
- `link: "/projects/ai-trading"` → 文件：`app/projects/ai-trading/page.tsx`
- `link: "/projects/data-viz"` → 文件：`app/projects/data-viz/page.tsx`

### 步骤4：准备项目图片

将图片放入 `/public/images/projects/` 目录：

```
public/images/projects/
├── ai-trading.jpg           # 项目1封面（用于列表页）
├── ai-trading-1.jpg         # 项目1详情图1
├── ai-trading-2.jpg         # 项目1详情图2
├── data-viz.jpg             # 项目2封面
├── data-viz-1.jpg           # 项目2详情图1
└── ...
```

**图片规范：**
- 封面图：1200 x 800 像素（3:2 比例）
- 详情图：1600 x 900 像素（16:9 比例）

## 🔗 路径对应关系总结

| 详情页文件路径 | 访问URL | 项目列表中的 link 字段 |
|--------------|---------|---------------------|
| `app/projects/ai-trading/page.tsx` | `/projects/ai-trading` | `link: "/projects/ai-trading"` |
| `app/projects/data-viz/page.tsx` | `/projects/data-viz` | `link: "/projects/data-viz"` |
| `app/projects/mobile-app/page.tsx` | `/projects/mobile-app` | `link: "/projects/mobile-app"` |

**规则：** 
- 文件夹名 = URL路径 = link字段值（去掉 `/projects/` 前缀）
- 例如：文件夹 `ai-trading` → URL `/projects/ai-trading` → link `"/projects/ai-trading"`

## 💡 快速创建新项目的步骤

1. **复制示例文件夹**
   ```
   复制：app/projects/example/
   粘贴为：app/projects/你的项目名/
   ```

2. **修改 page.tsx 内容**
   - 修改标题、日期、标签
   - 修改章节内容
   - 修改图片路径

3. **添加到项目列表**
   ```typescript
   // 在 components/projects-section.tsx 中添加
   {
     title: "项目标题",
     description: "项目描述",
     tags: ["标签1", "标签2"],
     year: "2024",
     featured: true,
     image: "/images/projects/封面.jpg",
     link: "/projects/你的项目名",  // 与文件夹名对应
   }
   ```

4. **准备图片**
   - 放入 `/public/images/projects/`
   - 命名规范：`项目名.jpg`, `项目名-1.jpg`, ...

5. **测试**
   - 刷新页面
   - 点击项目卡片
   - 检查跳转是否正确
